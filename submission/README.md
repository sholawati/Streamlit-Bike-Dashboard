# Dashboard Penyewaan Sepeda

Dashboard ini menampilkan penyewaan sepeda berdasarkan musim dan cuaca.

## Cara Menjalankan

1. Install semua kebutuhan:

```bash
   pip install -r requirements.txt
```

2. Jalankan Streamlit:

```bash
   streamlit run dashboard/dashboard.py
```

## Dataset

Data asli: `data/day.csv`  
Data olahan: `dashboard/main_data.csv`
